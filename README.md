# msf-termux
Install Metasploit easily on Termux


# Installation

    git clone https://github.com/Deadpool2000/msf-termux

    cd msf-termux
    
    chmod +x install.sh
    
    chmod +x msfconsole
    
    chmod +x msfvenom

    ./install.sh
